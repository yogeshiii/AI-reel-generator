# AI-reel-generator
AI Reel Generator is a Python-based application designed to streamline content creation by converting user-uploaded images and text into dynamic, vertical video reels. It supports automatic text-to-speech conversion, seamless image transitions, and synchronized audio integration—making it ideal for social media, marketing, and storytelling.
In this program i have used ELEVENLABS for audio generation and ffmpeg to create seamless reels.
To run this program you have to make sure:
1) rum main.py and create reel by adding the image and text
2) then, simply you have to run the generate_proccess wich wiil first convert your text to audio(make sure to use your api key) file and then create the reel.
3) you will find the created reel in static/reel folder. THANKYOU!
